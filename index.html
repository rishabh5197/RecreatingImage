What's the difference ?
HTML
Hyper Text Markup Language
Creating the structure
  • Controls the layout of the content
  • Provides structure for website
  • The fundamental building block of any webpage
CSS
Cascading Style Sheet
Stylize the website
  • Applies styles to the web page elements
  • Targets various screen sizes to make pages responsive
  • Primarily handles the "look and feel" of web page
Js
Java Script
Increase Interactivity
  • Adds interactivity to a web page
  • Handles Complex function and Features
  • Programmatic code which enhances functionality
